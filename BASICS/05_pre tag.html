As you can see here, the text in pre tag is exactly same as we have written in our code. and it is also showing the spaces and line breaks as well. so, this is the main purpose of pre tag. 

            Spaces: multiple spaces are kept exactly as written
            Line breaks: each new line is preserved
            Tabs: retained as entered
            Indentation: remains unchanged
            Whitespace: not collapsed or normalized
            Font: typically rendered in a monospace font by default
