Selectors in CSS

These allow any user to add styles in any element of html

  1. Universel Selectors

    2. It target the whole html document

    we use * for innitilizing it

  2. Class Selectors

    3. Class selector is used in CSS to select and style multiple HTML elements that have the same class attribute.

    we use . for innitilizing it

  3. ID selector

    4. ID selector in CSS is used to select and style a single unique HTML element with a specific id attribute.

    we use # for innitilizing it

  4. ELement selector

    5. Element selector in CSS is used to select and style all HTML elements of a specific type (tag name).

    we particular tag for innitilizing it

  5. Group selector

    6. Group selector in CSS is used to apply the same styles to multiple selectors at once by separating them with commas (,).

    we use particular tags seperated by comma

  6. Child selector

    7. Child selector in CSS is used to select and style direct child elements of a parent element using the > symbol.

    we use > for innitializing it

  7. Descendent selector

    8. Descendant selector in CSS is used to select and style all elements inside a parent element (at any level) using a space between selectors.

    we use space b/w precedent & Descendant for innitializing it
